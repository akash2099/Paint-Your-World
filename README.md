# PAINT YOUR WORLD Android Application :art:
A painting cum whiteboard app for your wide imaginations


## Features :fire:

- DRAW

- CHANGE BRUSH COLOR

- CHANGE BRUSH SIZE

- CHANGE BRUSH TYPE (NORMAL, BLUR)

- REDO, UNDO

- SAVE TO GALARY


## Framework Used :star2:
